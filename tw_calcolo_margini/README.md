# TW Calcolo Margini

Modulo Odoo 17 per il calcolo dei margini mensili aziendali.

- Calcolo automatico dei pagamenti ai fornitori
- Totale buste paghe mensili

## Installazione

1. Copiare il modulo nella cartella addons
2. Aggiornare la lista delle app
3. Installare "TW Calcolo Margini"

## Utilizzo

Aprire l'app "Calcolo Margini" dal menu principale e creare un nuovo record specificando il periodo di riferimento.
