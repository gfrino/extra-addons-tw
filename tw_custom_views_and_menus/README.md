# TW Custom Views and Menus

This module contains custom views and menus modifications for TW.

## Features

* Reorder dashboard menu items to show "La mia bacheca" as the first menu item
* Make "La mia bacheca" the default view when opening the Dashboard module

## Installation

1. Copy this module to your addons path
2. Update the module list
3. Install the module from the Apps menu

## Dependencies

* base
* spreadsheet_dashboard_oca

## Configuration

No additional configuration needed. The menu reordering is applied automatically upon installation.

## Usage

After installation, when you open the Dashboard module, it will automatically open the "La mia bacheca" view, which is now positioned as the first menu item.
