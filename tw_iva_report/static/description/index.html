IVA Report - VAT Amount in Payments ==================================== Display VAT/IVA amounts from invoices in payment records with automatic multi-currency conversion. Features -------- * Display VAT amounts in payment records * Multi-currency support with automatic conversion to company currency (CHF) * Form view integration with clear labels * Tree view with optional columns and sum totals * Works for both customer payments and vendor payments * Automatic calculation from reconciled invoices and bills Technical --------- * Adds computed fields for VAT amount in original and company currency * Stored fields for optimal performance * Automatic currency conversion using invoice date rates * Works with reconciled_invoice_ids and reconciled_bill_ids Installation ------------ 1. Install the module from Apps 2. No configuration needed 3.
