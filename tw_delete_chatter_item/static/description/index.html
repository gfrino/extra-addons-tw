Clean Up Your Chatter with Confidence

This powerful module allows authorized users to delete any message, note, or activity from the chatter. Perfect for maintaining clean, professional, and relevant communication threads across your entire Odoo system.

🔐 Permission-Based Access

Only authorized users with the "Delete Chatter Messages" group can delete messages. Complete control over who can clean up discussions.

Quick Setup Guide

  1. Install the module from Odoo Apps
  2. Go to Settings > Users & Companies > Users
  3. Select a user and edit their profile
  4. Enable "Delete Chatter Messages" group under Access Rights
  5. Save and the user can now delete chatter messages!
